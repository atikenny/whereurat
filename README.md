# whereurat
Add a bar to show where your scroll is on the top of the page

# usage
include the script

run: whereURAt.add()

# configuration
whereURAt.add(config);

example config:

{
    barId: 'whereURAt-bar',
    barColor: '#298AD9',
    barHeight: '2px',
    animate: true,
    animationSpeed: 0.1,
    zIndex: '999999'
}
